# 🏧 ATM Machine Simulation using Python

## 📖 Overview

This project is a simple ATM Machine Simulation developed in Python using conditional statements (`if-elif-else`) and user input handling. The program allows users to perform basic banking operations such as withdrawing money, depositing money, checking account balance, and changing their ATM PIN.

The project was created to practice Python fundamentals and menu-driven programming.

---

## ✨ Features

- 🔐 PIN Authentication
- 💸 Withdraw Money
- 💰 Deposit Money
- 🔄 Change ATM PIN
- 📊 Check Account Balance
- 🚪 Exit Option
- ⚠️ Invalid Choice Handling

---

## ▶️ How to Run

1. Install Python 3.
2. Download the project files.
3. Open the project in Python IDLE or any Python IDE.
4. Run the file:

```bash
python atm_machine.py
```

5. Enter the correct PIN to access ATM services.
